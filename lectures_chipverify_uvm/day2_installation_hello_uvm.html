표준화된 testbench 언어

UVM은 특정 프로젝트의 임시 testbench 스타일을 공통 vocabulary와 class library로 정리한 방법론입니다.

UVM installation and Hello UVM에서 볼 코드 모양

Hello UVM의 핵심은 simulator entry point와 UVM class world가 `run_test()`로 만나는 지점입니다.

import uvm_pkg::*;
`include "uvm_macros.svh"

module tb_top;
  initial begin
    // HDL top에서 UVM test class를 시작한다.
    run_test("smoke_test");
  end
endmodule
