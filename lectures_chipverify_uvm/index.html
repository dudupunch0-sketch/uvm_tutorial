how to use

학습 루프

1. Workbook 먼저

한국어 목표와 mental model을 먼저 읽고 오늘 볼 UVM 부품의 위치를 잡습니다.

2. ChipVerify 원문 확인

각 Day의 source card에서 원문을 열어 class/API 이름과 예제 모양을 확인합니다.

3. LLM으로 복습

prompt box에 내 요약과 질문을 넣어 개념을 다시 설명받고, quiz/practice로 닫습니다.
