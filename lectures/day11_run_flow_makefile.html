mental model

오늘 잡을 개념

compile target직접 compile하는 top/test 파일과 include 파일을 구분한다.
+incdirinclude 파일 검색 경로를 simulator에 알려준다.
runtime plusarg+UVM_TESTNAME, +UVM_CONFIG_DB_TRACE가 실행 때 UVM 동작을 바꾼다.
