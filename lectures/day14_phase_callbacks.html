Advanced 1: phase callback

UVM phase callback으로 기존 component 코드를 직접 고치지 않고 phase 동작을 관찰/확장하는 방법을 본다.

오늘의 완료 기준callback을 쓰는 이유를 한 문장으로 설명한다.
학습 루프개념 -> 실제 파일 -> 흐름 추적 -> 퀴즈 -> 작은 실습 순서로 진행합니다.
